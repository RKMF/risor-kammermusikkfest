# Translation Rules: Norwegian ↔ English

This guide defines translation conventions for Risør Chamber Music Festival content.

## Core Principles

1. **Norwegian is source of truth** - Always translate from Norwegian to English
2. **Translate meaning, not only words** - Do the work, don't take the lazy word-for-word route
3. **Match tone and register** - Marketing stays warm/playful, program notes stay formal
4. **Preserve factual accuracy** - Dates, names, places must be identical
5. **Idiom honesty** - If no good equivalent exists, explain and suggest alternatives. Never invent.

## Language-Specific Rules

### Punctuation

| Element | Norwegian | English |
|---------|-----------|---------|
| Quotation marks | «tekst» or "tekst" | "text" |
| Dialogue | – Hei, sa hun. | "Hi," she said. |
| Apostrophe | Rare (Oles bil) | Common (Ole's car) |
| Em-dash | – (with spaces) | — (no spaces) or – |
| Ellipsis | ... or … | … |

### Capitalization

| Element | Norwegian | English |
|---------|-----------|---------|
| Weekdays | mandag, tirsdag | Monday, Tuesday |
| Months | januar, februar | January, February |
| Nationalities | norsk, engelsk | Norwegian, English |
| Languages | norsk, engelsk | Norwegian, English |
| Titles/Roles | kunstnerisk leder | Artistic Director |
| Musical keys | C-dur, a-moll | C Major, A Minor |

**Musical key rule:** Norwegian uses lowercase "moll", capitalized "dur". English capitalizes both.

### Dates & Numbers

| Element | Norwegian | English |
|---------|-----------|---------|
| Date format | 24. juni 2025 | June 24, 2025 |
| Time format | kl. 19:00 | 7:00 PM or 19:00 |
| Thousands separator | 1 000 or 1.000 | 1,000 |
| Decimal separator | 3,14 | 3.14 |

## Structural Sync Rules

When translating, ensure:

1. **Same structure** - EN must have identical components as NO
2. **Same order** - Components appear in same sequence
3. **Same references** - Images, links, artist references identical
4. **Same facts** - Dates, venues, names never differ

### Idioms

Find equivalent expressions, not literal translations:

| Norwegian | NOT this | Instead |
|-----------|----------|---------|
| før jula spiser deg helt opp | before Christmas eats you up | before the holiday rush gets you |
| Det lover vi. | We promise it. | We promise. / Count on it. |

If no good equivalent exists, explain the meaning and suggest alternatives.

## Third-Party Quote Translation

When translating content that contains quotes from third parties (interview quotes, reviews, etc.):

### Principle: Translate Transparently

Third-party quotes in the Norwegian content that were originally in another language should be handled consistently in the English version.

### Scenarios

| Original Quote Language | In Norwegian Content | In English Content |
|------------------------|---------------------|-------------------|
| Norwegian | Keep as-is | Translate to English |
| English | Translate to Norwegian | Keep original English |
| Other (German, French, etc.) | Translated + attributed | Use same translation + attribution |

### Attribution Preservation

If a quote in the Norwegian content includes a translation attribution:
- `"Sitat." [oversatt fra tysk]`

The English version should maintain equivalent attribution:
- `"Quote." [translated from German]`

### Example

**Norwegian source:**
> "Jeg vil være en ærlig kunstner." [oversatt fra tysk] - Die Zeit

**English translation:**
> "I want to be an honest artist." [translated from German] - Die Zeit

## Tone Matching

| Context | Norwegian Tone | English Equivalent |
|---------|---------------|-------------------|
| Marketing copy | Warm, playful, "du/dere" | Warm, playful, "you" |
| Program notes | Formal, educational | Formal, educational |
| Ticket info | Direct, clear | Direct, clear |
| Artist bios | Personal, admiring | Personal, admiring |

The English translation should feel native, not translated. Match the warmth and personality of the Norwegian source.

## Quality Checks

Before finalizing translation:

- [ ] All facts identical (dates, names, places)
- [ ] Same number of components/sections
- [ ] Music terminology follows conventions
- [ ] Punctuation follows English rules
- [ ] Capitalization follows English rules
- [ ] Tone matches source register
- [ ] No literal translations of idioms
- [ ] Third-party quotes: attribution preserved (e.g., [translated from German])
- [ ] Field limits respected
