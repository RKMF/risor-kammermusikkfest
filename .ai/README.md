# AI Workflow Assets

This folder holds repo-owned workflow assets that are intended to work across AI tools.

## Structure

- `specialists/` - Domain-specific expert briefs for focused technical work
- `workflows/` - Repeatable operating procedures for common tasks
- `instructions/` - Cross-cutting rules, style guides, and reference material

## Usage

- Use `specialists/` when a task needs deep expertise in a specific domain.
- Use `workflows/` when following a defined process such as release, translation, or preparation.
- Use `instructions/` for shared standards that apply across tasks.
