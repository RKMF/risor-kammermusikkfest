# Norwegian Writing Style Guide

This guide defines the writing style for Risør Chamber Music Festival content. All Norwegian content should follow these patterns.

## Core Voice

**Warm, confident, personal** - Write as if speaking directly to a friend who loves music.

## Style Traits

| Trait | Pattern | Example |
|-------|---------|---------|
| Confident claims | Bold superlatives, no hedging | "verdens mest kjente", "verdens viktigste" |
| Direct reader address | Personal, conversational | "du", "dere", "deg" |
| Inclusive "we" | Festival as community | "vi", "vår", "oss" |
| Playful wit | Unexpected imagery, wordplay | "før jula spiser deg helt opp", "finurlig sidestilling" |
| Short sentences | Punchy, rhythmic endings | "Det lover vi." "Midt under sola i Risør." |
| Two-sentence structure | Setup + payoff | First sentence establishes, second delivers |
| Rhetorical questions | Engage reader curiosity | "hva velger årets artister når..." |
| Contrasts | Juxtaposition for effect | "sommer for den ene... vinter hos den andre" |
| Cultural references | Norwegian literary nods | "Bjørnstjerne Bjørnson valgte seg april" |
| Promise/reassurance | Build trust, warmth | "Det lover vi" |

## Sentence Rhythm

1. **Opening hook** - Bold claim or intriguing setup
2. **Development** - Brief elaboration or contrast
3. **Landing** - Short, punchy conclusion often with "du/dere" or a promise

### Example Breakdown

> *"En finurlig sidestilling av Vivaldi og Piazzolla."* [Hook - intriguing concept]
> *"For når det er sommer for den ene, er det vinter hos den andre."* [Development - explains the contrast]
> *"Midt under sola i Risør."* [Landing - grounds it locally, short punch]

## Bindeord (Conjunctions)

Use connectors purposefully to create flow and rhythm.

### Konjunksjoner (Coordinating)

Connect equal elements - use sparingly for punchy prose:

| Word | Use | Example |
|------|-----|---------|
| og | Connect words/phrases | "Vivaldi og Piazzolla" |
| men | Contrast | "kort, men intens" |
| eller | Alternative | "konsert eller festival" |
| for | Reason (literary) | "for når det er sommer..." |
| så | Result | "så det blir en fin kveld" |

### Subjunksjoner (Subordinating)

Introduce dependent clauses - good for development sentences:

| Word | Use | Example |
|------|-----|---------|
| at | That-clauses | "vi lover at det blir fint" |
| når | When/timing | "når festivalen starter" |
| fordi | Because | "fordi musikken fortjener det" |
| hvis | If/conditional | "hvis du vil oppleve noe unikt" |
| dersom | If/in case (formal) | "dersom du ønsker billetter" |
| selv om | Even though | "selv om sommeren er kort" |
| mens | While | "mens sola skinner" |
| derimot | However/on the other hand | "Vivaldi derimot valgte høsten" |
| likevel | Nevertheless | "likevel var det verdt det" |
| dessuten | Besides/moreover | "dessuten får du oppleve..." |
| enten...eller | Either...or | "enten konsert eller omvisning" |

### Og/å-regelen

A common error in Norwegian. See [Språkrådet: Å eller og](https://sprakradet.no/godt-og-korrekt-sprak/rettskriving-og-grammatikk/a-eller-og/) for the authoritative reference.

| Word | Function | Example |
|------|----------|---------|
| **og** | "and" - connects things | "epler og appelsiner" |
| **å** | "to" - before verbs (infinitive) | "å spise", "å oppleve" |

**Test:** Can you replace it with "and"? Use **og**. Does it come before a verb? Use **å**.

| Correct | Wrong |
|---------|-------|
| "Jeg liker **å** spise" | ~~"Jeg liker og spise"~~ |
| "spise **og** drikke" | ~~"spise å drikke"~~ |
| "for **å** sikre deg billetter" | ~~"for og sikre deg billetter"~~ |
| "Vivaldi **og** Piazzolla" | ~~"Vivaldi å Piazzolla"~~ |

## What to AVOID

- Formal/stiff language
- Passive voice
- Lengthy explanations
- Hedging words ("kanskje", "muligens", "nok", "vel")
- Corporate-speak
- Generic marketing phrases

## Real Examples from Festival Content

**Event excerpt:**
> "Verdens mest kjente elskende har gitt oss mer musikk enn vi klarer å fremføre. Derfor har vi valgt ut våre favoritter fra blant annet Prokofjev og Bernstein."

**Artist excerpt:**
> "En av verdens viktigste klassiske trompetister, og vår kunstneriske leder. Tine har løftet Risør kammermusikkfest tilbake til toppen, og det får du glede av."

**Homepage:**
> "Trykk på lenken ovenfor for å sikre deg billetter før jula spiser deg helt opp. Det blir like fint som det alltid er. Det lover vi."
